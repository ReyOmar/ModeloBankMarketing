# Modelo de Clasificación — Bank Marketing

## Autores
- Jostein Alexander Farfán Orjuela  
- Rey Omar Esguerra Ramírez

## Descripción
Proyecto de Ciencia de Datos para construir un modelo de clasificación que identifique qué clientes tienen mayor probabilidad de suscribirse a la campaña bancaria. Incluye limpieza y preparación del dataset, selección y evaluación de modelos, y análisis de resultados para apoyar decisiones de marketing.

## Dataset
- Nombre: **Bank Marketing**
- Link: https://www.kaggle.com/datasets/henriqueyamahata/bank-marketing/data
- Objetivo: Predecir la suscripción de clientes a la campaña

## Objetivos

### General
Desarrollar un modelo de clasificación para identificar qué clientes tienen mayor probabilidad de suscribirse a la campaña del banco, con el fin de optimizar recursos y mejorar la efectividad del marketing, mediante la preparación del dataset, el modelado y el análisis de resultados.

### Específicos
- Limpiar y preparar el dataset para asegurar datos consistentes y variables listas para el modelado.
- Seleccionar el modelo de clasificación más adecuado según métricas de desempeño relevantes para el problema.
- Analizar la importancia de las variables y los resultados obtenidos para generar conclusiones útiles.
